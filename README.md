# Fitness Landing Page

This is a responsive landing page built as a portfolio project for a fitness/gym website.

## Features
- Fully responsive design (mobile & desktop)
- Modern UI with dark theme
- Smooth animations and interactive navbar
- Services and pricing sections
- Contact form layout

## Built With
- HTML5
- CSS3
- JavaScript (vanilla)

## Purpose
This project was created to demonstrate front-end skills for my portfolio and freelance platforms like Fiverr.
